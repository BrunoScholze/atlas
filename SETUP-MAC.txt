SETUP DO ATLAS NO MAC
=====================

PRE-REQUISITOS
--------------
Abrir o Terminal e instalar:

  # Node.js — baixar em https://nodejs.org ou via Homebrew:
  brew install node

  # Claude Code CLI
  npm install -g @anthropic-ai/claude-code

  # Logar no Claude (necessario para o agente rodar)
  claude login


1. CLONAR O ATLAS
-----------------
  git clone <url-do-atlas> ~/atlas


2. CLONAR O APP DENTRO DE repos/
---------------------------------
  git clone <url-do-app-minha-producao> ~/atlas/repos/app-minha-producao


3. INSTALAR DEPENDENCIAS DO SERVIDOR
--------------------------------------
  cd ~/atlas/server
  npm install


4. CRIAR O .env DO SERVIDOR
----------------------------
  touch ~/atlas/server/.env

Abrir o arquivo e colar (substituir <seu-usuario> pelo resultado de `whoami`):

  PORT=3000
  CONTEXT_PATH=/Users/<seu-usuario>/atlas
  OUTPUT_PATH=/Users/<seu-usuario>/atlas/output.txt
  REPO_PATH=/Users/<seu-usuario>/atlas/repos/app-minha-producao
  TEMP_PATH=/Users/<seu-usuario>/atlas/temp


5. CRIAR A PASTA temp
----------------------
  mkdir ~/atlas/temp


6. RODAR O SERVIDOR
--------------------
  cd ~/atlas/server
  node index.js

Voce deve ver no terminal:
  Servidor rodando em http://localhost:3000

Deixe esse terminal aberto enquanto usa o plugin. Para parar: Ctrl+C.


7. CARREGAR O PLUGIN NO CHROME
--------------------------------
  1. Abrir chrome://extensions
  2. Ativar Modo do desenvolvedor (canto superior direito)
  3. Clicar em Carregar sem compactacao
  4. Selecionar a pasta ~/atlas/plugin


USO NO DIA A DIA
-----------------
Toda vez que abrir o Mac, antes de usar o plugin basta rodar:

  cd ~/atlas/server && node index.js
