**Obs.:** `toISOString()` retorna a data em UTC. Se o app rodar em fuso negativo (ex: BRT -03:00), o horário no componente aparece correto, mas se o PO-UI fizer parse sem ajuste de timezone pode exibir o dia anterior. Se isso ocorrer, troque por:

```typescript
const today = new Date();
const iso = `${today.getFullYear()}-${String(today.getMonth()+1).padStart(2,'0')}-${String(today.getDate()).padStart(2,'0')}T00:00:00.000Z`;
```

Para a maioria dos casos `toISOString()` já funciona sem problema.
